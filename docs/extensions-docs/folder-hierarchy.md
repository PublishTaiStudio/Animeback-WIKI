### Extension folder hierarchy

here's how to organize your folder to create your extension!
```
.AnimeBack
   |
   |->extensions
   |    |
   |    |->extension-name
   |    |    |
   |    |    |->pages
   |    |    |    |
   |    |    |    |->css
   |    |    |    |    |
   |    |    |    |    |->index.css
   |    |    |    |
   |    |    |    |->js
   |    |    |    |    |
   |    |    |    |    |->index.js
   |    |    |    |
   |    |    |    |->index.html
   |    |    |    |
   |    |    |
   |    |    |->extension.json
   |    |    |->main.js
   |    |    |->icon.png
   |    |    |
```
