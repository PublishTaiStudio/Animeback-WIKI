
### Hiérarchie du dossier de l'extension

voici comment organiser votre dossier pour crée votre extension !
```
.AnimeBack
   |
   |->extensions
   |    |
   |    |->extension-name
   |    |    |
   |    |    |->pages
   |    |    |    |
   |    |    |    |->css
   |    |    |    |    |
   |    |    |    |    |->index.css
   |    |    |    |
   |    |    |    |->js
   |    |    |    |    |
   |    |    |    |    |->index.js
   |    |    |    |
   |    |    |    |->index.html
   |    |    |    |
   |    |    |
   |    |    |->extension.json
   |    |    |->main.js
   |    |    |->icon.png
   |    |    |
```
