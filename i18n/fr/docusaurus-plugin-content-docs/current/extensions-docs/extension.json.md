### fichier extension.json
c'est dans le fichier extension.json que vous devez renseigner le nom du fichier `main.js`
```json
{
    "name": "Example Extension",
    "author": "Tai Studio",
    "version": "1.0.0",
    "main": "main.js"
}
```
